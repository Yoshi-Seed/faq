# 救急×お薬手帳 チェッカー（MVP）

搬送中の**10秒判断**を助ける、オフライン対応の**超軽量・静的サイト**です。
GitHub Pages にアップロードするだけで使い始められます。

## 使い方（GitHub Pages）
1. このフォルダ一式をリポジトリにコミット（例: `ems-meds-check`）。
2. GitHub → **Settings** → **Pages** → Source を `main` / `/ (root)` に設定。
3. 数十秒後、公開URL（`https://<user>.github.io/ems-meds-check/`）でアクセスできます。
4. スマホで開いたら「ホーム画面に追加」→ **PWA** としてオフライン動作します。

## 機能
- 患者のお薬（一般名/販売名/カナ/英字）を検索→チップ追加
- 「これから投与」薬（ニトロ/アドレナリン/アスピリン/サルブタモール/ブドウ糖）をタップ
- 救急現場で重要な**相互作用ルール**のみを色分け表示（Red/Orange/Yellow）
- **データはローカル**にキャッシュ（電波なしでも動作）

## 収録ルール（初期）
- PDE‑5阻害薬 × ニトログリセリン：**禁忌**（24h/48h）
- β遮断薬内服下のアナフィラキシー：**アドレナリン反応低下→グルカゴン検討**
- 抗凝固/抗血小板 × アスピリン：**出血リスク↑**（IM回避/圧迫）
- サルブタモール × β遮断薬：**反応低下** / × 利尿薬：**低K**→不整脈注意
- ブドウ糖（IV） × インスリン/スルホニル尿素：**再低血糖**に注意

## データの追加
- `data/drugs.js` に薬の同義語やクラス旗を追加
- `data/rules.js` にルールを追加（`planned_any_of` と `patient_classes_any_of` を組み合わせ）
